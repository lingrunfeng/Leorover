## 🆘 如果遇到问题

### 问题1：SSH连接失败
```bash
# 重新ping测试
ping 10.3.14.59

# 如果ping不通，重新配置网络
cd ~/mycobot/real_robot_tests
bash setup_network_connection.sh
```

### 问题2：找不到launch文件
可能是树莓派上的ROS2工作空间没有source，在树莓派上运行：
```bash
source ~/colcon_ws/install/setup.bash
ros2 launch mycobot_280pi slider_control.launch.py
```

### 问题3：机械臂不动
- 检查电源是否打开
- 检查伺服电机是否通电（手动轻轻移动，如果很松说明没通电）
- 重启机械臂电源
